IT Support System Client v1.1.0
============================

This is the desktop client application for the IT Support System.

Requirements:
------------
- Java 17 or higher
- Access to the IT Support System server

Installation:
------------
1. Download and extract the support-client.jar file
2. Make sure you have Java 17 or higher installed
3. Run the application using: java -jar support-client.jar

Default Credentials:
------------------
Admin User:
- Username: admin
- Password: admin

Regular User:
- Username: employee
- Password: employee

Features:
--------
- Create and manage IT support tickets
- Track ticket status and updates
- Priority-based ticket handling
- Comments system with threaded discussions
- Modern and intuitive interface

What's New in v1.1.0:
-------------------
- Added comments system for tickets
- Real-time comment updates
- User attribution for comments
- Chronological comment history
- Improved UI for ticket details

For more information and server setup instructions, visit:
